# Infinite Matrix

A first-person endless tunnel game built in **Unreal Engine 5.7**, following the [Kodeco “How to Create a Simple Game in Unreal Engine 4”](https://www.kodeco.com/454-how-to-create-a-simple-game-in-unreal-engine-4) tutorial (adapted for UE5).

## Requirements

- **Unreal Engine 5.7** (or the version specified in `InfiniteMatrix.uproject`)

## Project structure

- **Content/InfiniteMatrix/** – Blueprints, maps, materials, meshes, textures, and UI
- **Config/** – Engine and project settings
- **InfiniteMatrix.uproject** – Project file (open this in the Unreal Editor)

## Getting started

1. Open `InfiniteMatrix.uproject` in Unreal Editor (double-click or from Epic Games Launcher).
2. Play from **TutorialMap** (default map).

## Clone and LFS

This repo uses **Git LFS** for `.uasset` and `.umap` files. After cloning, run:

```bash
git lfs pull
```

(If LFS wasn’t installed before clone, install it and run `git lfs install`, then `git lfs pull`.)
